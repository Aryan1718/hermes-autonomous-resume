# Backend Endpoints

This section documents the backend API endpoints required by the Hermes resume system.

The goal of this section is to explain what each endpoint does, what data it should accept, and how the scraper agent and resume agent use it during the workflow.

You can use this section to implement your own backend regardless of stack, as long as the same API contracts are available to the agents.
