# Extraction Workflow Checklist

Phase-by-phase checklist for the agent to follow during JD extraction.

## Phase 1: Read

- [ ] Extracted Layer 0 hard screens before deeper analysis
- [ ] Read the JD three times (surface, behavioral, cultural)
- [ ] Diagnosed the "Why Now?"
- [ ] Categorized requirements: Must / Preferred / Filler
- [ ] Noted the seniority signals
- [ ] Identified the hidden stack
- [ ] Identified potential disqualifiers (Anti-Resume)

## Phase 2: Extract

- [ ] Listed all must-have keywords and tools
- [ ] Listed all verb-noun pairs from responsibilities
- [ ] Listed scope signals: ownership object, stakeholders, decision authority, pace, complexity
- [ ] Listed the cultural/tone signals
- [ ] Noted quantification trigger words
- [ ] Labeled explicit facts vs. inferences
- [ ] Produced the final extraction artifact

## Phase 3: Final Review

- [ ] Every signal has a confidence label (explicit / high / medium / low)
- [ ] Every signal has a priority label (A / B / C)
- [ ] Must-haves are distinguished from preferred and filler
- [ ] Verb-noun pairs are captured in JD's exact language
- [ ] "Why Now?" diagnosis is recorded in cultural_intent_signals
- [ ] Anti-Resume signals are recorded
- [ ] Open questions are flagged (ambiguous requirements, unclear scope)
